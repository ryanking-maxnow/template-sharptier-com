# Docs Hub

本目录用于约束 `template-sharptier-cms` 工程必须先参考文档再实施。

## 目录
- `project/`：项目级文档（README、开发指南、部署 SOP、问题追踪）
- `component-docs/nginx-docs/`：Nginx 组件文档
- `component-docs/postgresql-docs/`：PostgreSQL 组件文档
- `component-docs/payload-docs/`：PayloadCMS 组件文档

## 使用规则
1. 任何工程变更前，先定位对应 `skills/` 与本目录的文档。
2. 变更说明中应明确引用使用了哪些 skills/docs。
3. 未完成 skills/docs 对齐前，不做工程闭环交付。
